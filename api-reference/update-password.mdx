## POST /api/updatePassword/{id}

Updates an existing password entry. You can update any combination of fields (password, sistema, usuario).

### Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the password entry to update
</ParamField>

### Request Body

<ParamField body="password" type="string" optional>
  The new password value
</ParamField>

<ParamField body="sistema" type="string" optional>
  The new system or application name
</ParamField>

<ParamField body="usuario" type="string" optional>
  The new username
</ParamField>

### Request

```bash
curl -X POST http://your-domain.com/api/updatePassword/1 \
  -H "Content-Type: application/json" \
  -d '{
    "password": "newSecurePass456",
    "sistema": "Production Database",
    "usuario": "admin"
  }'
```

### Response

<ResponseField name="password" type="object">
  The updated password object
  
  <ResponseField name="id" type="integer">
    Unique identifier for the password entry
  </ResponseField>
  
  <ResponseField name="password" type="string">
    The updated password value
  </ResponseField>
  
  <ResponseField name="sistema" type="string">
    The updated system or application name
  </ResponseField>
  
  <ResponseField name="usuario" type="string">
    The updated username
  </ResponseField>
  
  <ResponseField name="status" type="integer">
    Status flag (1 = active, 0 = deleted)
  </ResponseField>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="status" type="integer">
  HTTP status code
</ResponseField>

### Response Codes

- **200** - Success - Password updated successfully
- **404** - Password not found

### Example Response (200 OK)

```json
{
  "password": {
    "id": 1,
    "password": "newSecurePass456",
    "sistema": "Production Database",
    "usuario": "admin",
    "status": 1
  },
  "message": "contraseña actualizada",
  "status": 200
}
```

### Example Response (404 Not Found)

```json
{
  "message": "contraseña no encontrada",
  "status": 404
}
```
